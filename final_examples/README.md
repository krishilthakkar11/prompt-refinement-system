# Final Examples - Multi-Modal Prompt Refinement System

This directory contains 5 diverse examples demonstrating different capabilities and scenarios of the prompt refinement system.

## Quick Overview

| Example | Input Type | Completeness | Key Feature |
|---------|-----------|--------------|-------------|
| [1. Detailed E-commerce](example1_detailed_ecommerce/) | Text | 1.00 | Perfect input handling |
| [2. Minimal/Vague](example2_minimal_vague/) | Text | 0.65 | Gap handling & assumptions |
| [3. Multi-modal Conflict](example3_multimodal_conflict/) | Text + Image + PDF | 0.67 | Conflict detection |
| [4. Document PRD](example4_document_prd/) | PDF | 1.00 | Document extraction |
| [5. Contradictory](example5_contradictory/) | Text | 0.87 | Logic validation |

## Structure

Each example folder contains:
- **Input file(s)**: Original input (text, image, PDF, or combination)
- **output.json**: Refined prompt structure generated by the system
- **README.md**: Description, expected vs actual results, what it demonstrates

## How to Use

### View Examples
Browse each folder to see inputs and corresponding outputs.

### Regenerate Examples
Run the generation scripts:
```bash
python generate_example1.py  # Detailed e-commerce
python generate_example2.py  # Minimal/vague
python generate_example3.py  # Multi-modal conflicts
python generate_example4.py  # Document-heavy
python generate_example5.py  # Contradictory requirements
```

### Test with Your Own Inputs
Use the Streamlit UI:
```bash
python -m streamlit run app.py
```
